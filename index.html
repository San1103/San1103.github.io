Java Project

As for my Java Project, I decided to build a 2D Platformer which took some inspiration from games such as Mario. There are multiple levels, each having its own unique theme with different Backgrounds and platform types. There are also enemies in the game which try to halt your progress throughout as well as other Objects which will kill the player. The player also has a laser-eyes as a projectile to damage Objects/Enemies. Most levels require parkour around the map, which could be quite challenging.
